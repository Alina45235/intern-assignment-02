# Bank Card Form

Форма ввода банковской карты с валидацией и режимом скрытия данных.

## 📝 Описание задания

Разработать экран формы ввода данных банковской карты с полной валидацией и режимом скрытия визуальных данных.

**Технологический стек:**
- Kotlin + Jetpack Compose
- MVVM архитектура
- Koin для DI
- Kotlinx Coroutines
- Kotlinx Serialization
- Ktor Client
